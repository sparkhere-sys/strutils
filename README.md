**Str**ing **Util**itie**s**

A header-only (for now) library that adds functions that help with managing strings for C++!

## Installation

Clone the repo:
```bash
git clone https://github.com/sparkhere-sys/strutils.git
cd strutils
```

Copy `include/StrUtils.hpp`
